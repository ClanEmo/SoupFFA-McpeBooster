This Plugin has now a MultiLang System with English, German Arabic and French, more comming soon.
Change the language easily in the config.yml

<br>
<br>

How to install:

First change the world in the config.yml

Second create a Sign with "SoupFFA" on the FirstLine

Click on the Sign and play with you friends :D

<br>
<br>

When a player has the Permission "soupffa.vip", the player gets a Vip Kit!

<br>

You can join SoupFFA directly per the "/soupffa join" Command or per Sign
